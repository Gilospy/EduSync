Data Architecture Explained

1. Retrieval

Data is extracted nightly from the LMS PostgreSQL DB and Focus Tracking API.

2. Processing

An Azure Data Factory pipeline transforms raw logs into a Star Schema (Fact_Attendance, Fact_Grades).

3. Calculation (DAX)

Power BI calculates metrics like Risk Score dynamically based on weighted averages of attendance and grades.

4. Embedding

This web app authenticates via Azure AD Service Principal to securely render the report without requiring user licenses.
